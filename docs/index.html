pgmock

An NPM module for mocking a connection to a PostgreSQL database.

pgmock
Version: 1.0.0
Author: Jason Favrod <mail@jasonfavrod.com>
Example
const pgmock = require('pgmock'),
pgMock = pgmock();

add

Add a query, it's value definitions, and response to the mock database.

add(query: string, valueDefs: array, response: object)
Parameters
query (string) An SQL query statement.
valueDefs (array) Contains the types of each value used in the query.
response (object) The simulated expected response of the given query.
Example
pgMock.add("SELECT * FROM employees WHERE id = $1", ['number'], {
    rowCount: 1,
    rows: [
        { id: 0, name: 'John Smith', position: 'application developer' }
    ]
});

drop

Remove a query from the mock database.

drop(query: string): boolean
Parameters
query (string) An SQL statement added with the add method.
Returns
boolean: true if removal successful, false otherwise.
Example
pgMock.drop("SELECT * FROM employees WHERE id = $1");
